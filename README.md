# Monte Viejo Landing

Static landing page ready for GitHub Pages.

## Local preview

Open `index.html` in a browser.

## Client content edits

Edit `content.js` for text, WhatsApp number, and image paths.

Upload replacement images to `assets/images/`.

See `CLIENT_GUIDE.md`.

## Publish on GitHub Pages

Use repository root as Pages source.

https://jamir-boop.github.io/monte-viejo/
